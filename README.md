# DeepFlower
通过迁移学习，在MobileNet上再次训练一个自己创建的花卉数据集，该数据集包括24种花卉，共一万多种图片，来自网络抓取。最后测试准确率达85%。借助于tensorfow Lite API，将其部署于android平台，开发出DeepFlower应用。
